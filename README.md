# DjangoBlogProject

Это веб-приложение на языке Python с использованием фреймворка Django. Проект представляет собой платформу для ведения блогов, дополненную функциями социальной сети и общим чатом для зарегистрированных пользователей.

## 🚀 Основные возможности

### Для всех посетителей:
* **Просмотр контента:** можно читать любые посты и комментарии к ним без регистрации.
* **Открытость:** доступ к информации о публикациях других авторов.

### Для зарегистрированных пользователей:
* **Управление постами:** создание новых записей, а также возможность редактировать или удалять свои публикации.
* **Работа с медиа:** поддержка загрузки и отображения фотографий в постах.
* **Система комментариев:** полноценное обсуждение под постами с возможностью изменять или удалять свои сообщения.
* **Общий чат:** раздел для мгновенного обмена сообщениями со всеми участниками сообщества.
* **Аккаунт:** функции регистрации, авторизации и выхода из системы.

## 🛠 Стек технологий
* **Python 3**
* **Django** (основной фреймворк)
* **SQLite** (база данных)
* **HTML5 / CSS3** (интерфейс)

## 📦 Как запустить проект у себя

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/ваш-логин/DjangoBlogProject.git](https://github.com/ваш-логин/DjangoBlogProject.git)
   cd DjangoBlogProject
Настройте виртуальное окружение:

Bash
python -m venv venv
source venv/bin/activate
Установите необходимые библиотеки:

Bash
pip install -r requirements.txt
Примените настройки базы данных:

Bash
python manage.py migrate
Запустите локальный сервер:

Bash
python manage.py runserver
Проект будет доступен в браузере по адресу: http://127.0.0.1:8000/
